# 🤖 NovaChat AI

A modern AI chatbot built with HTML, CSS, and JavaScript.

## ✨ Features

* 💬 Modern chat interface
* 🎨 Clean and responsive design
* ⚡ Fast performance
* ⌨️ Press **Enter** to send messages
* 🔄 Start a new chat
* 🤖 Simulated AI responses
* 📱 Works on desktop and mobile browsers
